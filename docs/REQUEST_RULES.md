# Правила составления заявок

## Содержание и тон

- Использовать официально-деловой тон
- Подробно описывать одну наблюдаемую проблему с признаками и последствиями
- Указывать практические последствия, только если они известны из ввода
- Добавлять раздел «Прошу:» с нумерованными пунктами
- Выбирать законодательство под тип проблемы:
  - Постановление № 290 — содержание общего имущества
  - Постановление № 354 — коммунальные услуги (отопление, вода)
  - СанПиН — санитарные нормы (плесень, сырость, чистота)
  - ЖК РФ — жилищные права
  - Постановление № 410 — газовое оборудование
- Обосновывать, почему выбранное постановление применимо
- Преобразовывать эмоции в наблюдаемые факты

## Ограничения

- Не придумывать место, причину, виновника или повреждения
- Не добавлять неподтверждённые обвинения
- Не добавлять вымышленные даты, номера договоров и актов
- Не выдумывать несуществующие номера постановлений
- Не создавать отдельный раздел «Правовое обоснование»
- Не просить предоставить письменный ответ по умолчанию

## Пример

**Ввод жителя:**
«Калитка во двор сломана, не закрывается, магнит болтается. В подъезд теперь кто угодно может зайти, страшно. Сделайте уже что-нибудь!»

**Заявка:**
«Прошу принять меры по ремонту общедомового имущества — электромагнитного замка на калитке № 1 (ведущей во двор), расположенной со стороны 6-го подъезда.

Крепление магнита ослабло, он шатается при попытке закрытия. Из-за этого между магнитом и ответной планкой образуется зазор, они не прилегают плотно друг к другу. Следствие неисправности: система удержания двери не срабатывает, калитка остаётся открытой или легко открывается без ключа доступа, что создаёт угрозу безопасности жильцов.

В соответствии с Постановлением Правительства РФ от 03.04.2013 № 290 прошу:

1. В кратчайшие сроки направить специалиста для диагностики неисправности.
2. Выполнить необходимые ремонтные работы: закрепить магнитный замок, отрегулировать его положение для обеспечения плотного прилегания либо заменить вышедшие из строя элементы крепления.
3. Проверить работоспособность системы контроля доступа после ремонта.»
