# ADR-0003: Провайдер-независимая граница LLM

## Статус

Accepted

## Контекст

Конкретный LLM-провайдер и модель ещё не выбраны. Доменный сценарий не должен
зависеть от SDK или формата ответа внешнего сервиса.

## Решение

Определить малый интерфейс `LlmGateway` в `core`. Реализации размещать в `llm`.
До выбора провайдера использовать отключённый адаптер, который возвращает
контролируемую инфраструктурную ошибку и не создаёт фиктивный результат.

## Последствия

Выбор провайдера можно сделать отдельным небольшим изменением. Проверки API уже
учитывают недоступность генерации. Граница не превращается в универсальную
систему плагинов.
